- About us

    Monoeは
        モノが良いと異議を申すものいいをあわせた造語です。
        良いものやことであり、それそのものが社会に影響を与える状態を多く生み出すことを目的とした組織です。
        そのため私達は常に良いものやよいことを求め、情報収集を行っています。
        そして良いものやよいことを世の中に浸透させていくためのアイデアを生み出し実行します。
